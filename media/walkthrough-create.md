## Create a TOGAF Vault

Use the `/new` command to generate a complete architecture vault:

```
@architect /new MyProject
```

This creates **28 template files** covering all TOGAF ADM phases:

- **Phase A** – Architecture Vision
- **Phase B** – Business Architecture
- **Phase C** – Information Systems
- **Phase D** – Technology Architecture
- **Governance** – Principles, decisions, risks, requirements
